Welcome to the 2022 E-Waste Code Jam

E-Waste is a growing problem accross the world. E-Waste can be anything from old batteries to your newest flat screen tv. Any of these products conatin components which need to be dealt with in a certain way in order to help our planet. Join our Code Jam to create a project to help spread awareness about this growing issue which impacts Everyone in the modern technological world!
